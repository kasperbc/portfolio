## About the game
---
This project is a demo I created for the mobile programming course, based on an idea I had about a slow-paced space-living simulation.

The demo implements the following things:
- Touch controls
    - Two-finger pinch to zoom
    - Tap to select an object
- Saving
- Push notifications
